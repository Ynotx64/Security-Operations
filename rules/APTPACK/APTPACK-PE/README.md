# APTPACK-PE

## Purpose
Persistence / Privilege Escalation focused pack for adversary behavior that modifies system state, establishes persistence, or leverages elevated mechanisms.

## Sub-branches
- `APTPACK-ANTI-BYPASS/` contains evasive persistence / privilege escalation patterns
- `APTPACK-CORR/` contains correlated PE detections
- `scenarios/` contains validation scenarios

## SOC relevance
This branch helps the SOC identify escalation and persistence tradecraft following initial execution.
