# APTPACK-CA

## Purpose
Command and Control focused pack for suspicious beaconing, check-ins, header anomalies, and outbound communications consistent with adversary-controlled infrastructure.

## Sub-branches
- `APTPACK-ANTI-BYPASS/` contains evasive or disguised C2 indicators
- `APTPACK-CORR/` contains correlated command-and-control detections
- `scenarios/` contains validation scenarios

## SOC relevance
This branch helps the SOC identify sustained malicious communications after compromise.
