# APTPACK-IA

## Purpose
Initial Access phase pack for adversary behavior associated with access establishment, suspicious authentication, and early web or network entry patterns.

## Sub-branches
- `APTPACK-ANTI-BYPASS/` contains evasive or disguised Initial Access indicators
- `APTPACK-CORR/` contains higher-confidence correlated Initial Access detections
- `scenarios/` contains validation scenarios for this phase

## SOC relevance
This branch helps the SOC identify suspicious entry behavior before or during foothold establishment.
