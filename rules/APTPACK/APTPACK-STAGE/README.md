# APTPACK-STAGE

## Purpose
Staging and exfiltration oriented pack for archive creation, transfer preparation, chunked movement, and outbound artifact delivery.

## Sub-branches
- `APTPACK-ANTI-BYPASS/` contains evasive exfil / staging patterns
- `APTPACK-CORR/` contains correlated staging detections
- `scenarios/` contains validation scenarios

## SOC relevance
This branch helps the SOC identify when an operator prepares or transfers collected data.
