# APTPACK

## Purpose
APTPACK is the working detection-content tree used to stage, validate, and organize custom SOC detections before GitHub publication and later deployment.

## Current active branches
- `demo/` contains operator-run demo or validation scripts
- `github-staging/` contains the exact files being prepared for GitHub
- `README.DISABLED.txt` is a legacy informational file kept at root

## SOC relevance
This tree is the bridge between local Wazuh content, campaign-specific detection development, and version-controlled detection engineering.
