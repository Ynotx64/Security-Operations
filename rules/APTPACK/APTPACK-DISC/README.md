# APTPACK-DISC

## Purpose
Discovery phase pack for host, account, network, and environment reconnaissance behavior.

## Sub-branches
- `APTPACK-ANTI-BYPASS/` contains evasive discovery patterns
- `APTPACK-CORR/` contains correlated discovery detections
- `scenarios/` contains validation scenarios

## SOC relevance
This branch helps the SOC detect operator reconnaissance that usually follows foothold establishment.
