# APTPACK-EXEC

## Purpose
Execution phase pack for adversary behavior involving malicious script execution, encoded commands, remote retrieval, and command interpreter use.

## Sub-branches
- `APTPACK-ANTI-BYPASS/` contains execution-related evasive patterns
- `APTPACK-CORR/` contains correlated execution detections
- `scenarios/` contains execution validation scenarios

## SOC relevance
This branch helps the SOC identify when a foothold begins actively executing attacker logic.
